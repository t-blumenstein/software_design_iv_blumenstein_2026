Note that the below code shows only the Observable aspect in order to demonstrate how the operatator chaining works with Observables. The actual code running this animation uses signals in order to create the animation of processing each data element at a time.The code using signals has been omitted below for simplicity, but can be seen in the actual source code for this file. 
          
      this._dataService.dataStream$.pipe(
        tap((elem) => {
          this.tapData.push(elem);          
        }),

        map((elem) => {    
          return elem * 2;
        }),

        filter((elem) => {        
          return elem % 2 == 0;
        })
      );
