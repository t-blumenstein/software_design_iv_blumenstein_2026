The filter() function takes in a lambda function by which to do the filtering. The function defined will be used as the predicate by which to filter by.
The below code shows only the Observable aspect in order to demonstrate how the filter() function works with Observables. The actual code running this animation uses signals in order to create the animation of processing each data element at a time.The code using signals has been omitted below for simplicity, but can be seen in the actual source code for this file.
